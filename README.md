# Rapid Response

Rapid Response is an iOS bedside reference app for rapid response, acute deterioration, and common inpatient emergencies.

It is built to give fast access to:

- condition-based emergency guides
- symptom-based bedside workups
- concise checklists and escalation steps
- medication, conversion, and scoring tools

## App Focus

Rapid Response is designed for hospital and acute care settings where quick access to practical clinical information matters.

Core areas include:

- cardiac arrest and ACLS workflows
- arrhythmias, shock, and respiratory failure
- neurologic, toxicologic, hematologic, and oncologic emergencies
- symptom-first pathways such as chest pain, syncope, headache, fever, and abdominal pain

## Features

- `By Condition` and `By Symptom` navigation
- favorites saved locally on device
- bedside checklists for selected high-acuity topics
- medication reference and quick tools
- clickable references and external resource links
- support/privacy pages for App Store distribution

## Privacy

Rapid Response does not require an account and does not include ads, analytics, or third-party tracking.

The app currently stores only limited local app data such as:

- saved favorites
- disclaimer acceptance state

If a user chooses to send feedback by email, the email is handled by Apple Mail or the user's configured mail app.
