Rapid Response Support

Rapid Response is a bedside clinical reference app for rapid response and acute care scenarios.

Need Help?

  • Open an issue on the GitHub repository.
  • Use the contact information listed on the App Store.

When Reporting a Problem

  • Device model
  • iOS version
  • App version
  • What you were trying to do
  • What happened instead
  • Screenshots, if available

Important Note

Rapid Response is a clinical reference tool and does not replace clinical judgment, institutional policy, or specialist consultation.
